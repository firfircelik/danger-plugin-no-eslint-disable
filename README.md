> prevent eslint disable

## Usage

Install:

```sh
yarn add danger-plugin-no-eslint-disable --dev
```

At a glance:

```js
// dangerfile.js
import noEslintDisable from 'danger-plugin-no-eslint-disable'

noEslintDisable()
```
